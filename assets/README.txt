Minky Homecare SG - asset override slots
=========================================

The site currently uses lightweight, self-contained SVG brand marks (drawn in
the HTML/CSS) so it deploys with zero external image dependencies. To swap in
your official Minky brand files, drop them into this /assets folder using the
exact filenames below - no code changes needed.

  logo-minky.svg / .png        Official minky homecare wordmark (header + footer)
  badge-made-in-uk.png         "Made in the UK" heart / UK Manufacturer Est 1941
  badge-anti-bacterial.png     "Silver Anti-Bacterial Technology" badge
  badge-royal-warrant.png      Royal Warrant crest
  og-image.png                 Social share image (1200x630) - a placeholder is
                               already here; replace with a branded version.
  favicon.svg                  Browser tab icon (placeholder provided).

If you provide the official PNG/SVG files, tell me and I will wire them into the
markup in place of the inline SVGs.
